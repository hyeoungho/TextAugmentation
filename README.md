# UIFtextaugmentation
UIF Text Augmentation Project

## Prerequisite

Below is the list of libraries that you need to install. Even though requirement.txt contains all the library list, you don't need to manually install all of them. 
  ```
  tensorflow(-gpu)==1.14
  tensorflow_hub
  pandas
  numpy
  sklearn
  ```
